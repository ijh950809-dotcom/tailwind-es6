04_자바스크립트 es6문법_class문법

클래스는 일종의 함수이지만 , function키워드 대신 class키워드를 사용한다.

속성은 constructor()메서드 내부에 할당한다.

class는 사실상 함수이기 때문에 함수선언과 변수선언식 둘다 선언이 가능하다.

    문법)
    class 클래스명(){
      constructor(){
        this.속성명 = 값;
      }
      메서드명(){

      }
    }
    -------------
    자바스크립트 함수 문법 
    function 함수명(매개변수1 , 매개변수2..){
      실행할 내용;
    }

    

    문법) 클래스호출 방법
    const 인스턴스명 = new클래스명();

class문법에서는 클래스명 작성시 무조건 '첫글자는 대문자'로 작성한다.

부모 클래스를 상속받는 자손 클래스

상속 : 부모 클래스의 멤버를 자식 클래스에세 물려주는것

부모클래스 : 상위클래스, 슈퍼클래스라고함

자손 클래스: 하위 클래스, 파생클래스라고함

부모와 자손을 나눠서 관리하는 이유는'재사용'해서 새로운 클래스를 만들기 때문에 '중복되는 코드를 보여주고 유지보수의 시간을 단축'하기 위함이다.
